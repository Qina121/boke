遇到的一些题

Wed, Apr 25, 2018

换主题时在目录中找不到文件？（解决方法） 换主题时在Hugo的文件里多创建了一个文件夹themes,将下载好的主题文件下载到其中，按文档提示操作，一两种主题为例（目前尝试的），一种是细致布局有exampleSites的，需要将其的里面的config文件复制的sites、blog文件中，覆盖原来的对应文件，将其他样式放入到themes里面，而另一种需要覆盖config，同同时执行命令cd themes 然后 hugo server -t material-design -w -D；操作即可

多个文档应该如何处理？（解决办法） 文档都是放在了blog中的content中，在创建文档的命令hugo new post/text2.md（text2随意命名）
